FredBoat is a bot made with JDA with both fun and convenience in mind.

Example commands:

  • ;;avatar <@username> - displays the users avatar
  • ;;brainfuck <code> [input] - executes brainfuck code
  • ;;lua <code> - executes Lua code in a sandboxed environment
  • ;;riot <text> - display your demands
  • ;;leet <text> - sound like a script kiddie
  • ;;dump [-q] <1-2000> - returns a hastebin dump of the last messages in the channel. Use -q to send in a DM instead.
  • ;;mal <search term> - searches MAL for an anime or a user account.

For a full list of commands, please message the bot or say ;;help in any channel the bot is in.

I'm always looking for new suggestions. Be sure to join our guild if you have any new ideas or need help!
