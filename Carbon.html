Music Bot is currently in Working Beta!

FredBoat is a bot made with JDA with both fun and convenience in mind. It now also comes as a music bot.

Example commands:

  • ;;avatar &/lt;@username&/gt; - displays the users avatar
  • ;;brainfuck &/lt;code&/gt; [input] - executes brainfuck code
  • ;;lua &/lt;code&/gt; - executes Lua code in a sandboxed environment
  • ;;riot &/lt;text&/gt; - display your demands
  • ;;leet &/lt;text&/gt; - sound like a script kiddie
  • ;;dump [-q] &/lt;1-2000&/gt; - returns a hastebin dump of the last messages in the channel. Use -q to send in a DM instead.
  • ;;mal &/lt;search term&/gt; - searches MAL for an anime or a user account.

Example music commands:

  • ;;play &/lt;URL&/gt; - plays a song from the URL, such as YouTube and Soundcloud.
  • ;;play &/lt;search terms&/gt; - returns up to 5 videos from youtube, from which you can choose to add to the queue.
  • ;;join - Joins the voice channel. The ;;play command will do this automatically.
  • ;;leave - Leave the current voice channel.
  • ;;skip - Skip a song in the queue.
  • ;;list - Get a list of songs currently playing.

For a full list of commands, please see the documentation, message the bot or say ;;help in any channel the bot is in.

I'm always looking for new suggestions. Be sure to join our server if you have any new ideas or need help!
