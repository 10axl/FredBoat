Welcome to FredBoat Docs

Welcome to the FredBoat docs. FredBoat is an open-source general-purpose bot developed in Java. The bot is based off of JDA and was created by (@Frederikam). You'll be able to get a link back to this website simply by invoking ;;help.

Setting up the bot

Getting FredBoat running on your server is simple. You can invite the bot by authenticating it with your account using this link. Bear in mind that you must have Manage Server permissions to be able to do this.
